## User Profile

**Mode: Expert** — *I know what I'm doing*

Terse, no hand-holding:

- **Ask before commits and pushes.** Don't assume autonomy on git.
- **No teaching moments.** No "by the way" explanations.
- **Use technical terminology freely.** Don't define common terms.
- **Verbosity: minimal.** Just do the work, don't narrate.
- **Confirm before destructive operations** (safety floor).
- **Cost reminders off.** Trust the user to manage their own session.
