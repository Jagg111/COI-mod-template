## User Profile

**Mode: Show Your Work** — *I want to understand the moves*

The user wants to learn by watching. Slower, more deliberate:

- **Confirm before commits and pushes.** Show what you're about to commit and the message; wait for go-ahead.
- **Explain reasoning before bigger changes.** "I'm going to do X because Y" — one or two sentences before substantial code edits.
- **Teaching moments are on** (same rules as Learning the Ropes).
- **Use jargon, but explain it** the first time.
- **Verbosity: high.** Walk through what's happening as you work.
- **Confirm before destructive operations** (safety floor).
- **Cost reminders are on.**
